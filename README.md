# T2_HI_Servidor

La finalidad del proyecto es aprender a implementar Spring security, para esta finalidad se
ha desarrollado una aplicación web que permita subir poemas a una base de datos a los
usuarios registrados, para poder visualizar estos poemas no hará falta estar registrado.
Hay dos tipos de usuarios, un usuario normal y un administrador, la única diferencia es que
el usuario administrador podrá tener visión de todos los usuarios registrados (y podrá
suspender su cuenta).
Por ello en este proyecto habrá 3 tablas en la base de datos, usuarios , poemas y
authentication.
